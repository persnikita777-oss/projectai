# ProjectAI.ru — Документация проекта

## Концепция

**ProjectAI.ru** — AI-студия полного цикла для малого и среднего бизнеса в России.
AI-ассистент на сайте автоматически принимает заявки, собирает ТЗ через диалог, генерирует план работ и коммерческое предложение.

**Миссия:** Сделать AI-разработку доступной для бизнеса любого размера — от ИП до среднего бизнеса.

**УТП (уникальное торговое предложение):**
- AI-ассистент вместо формы заявки — клиент получает предварительную оценку за 5 минут
- Скорость: MVP за 1-2 недели (vs 4-8 недель у конкурентов)
- Прозрачное ценообразование: от 150K руб
- Мультимодельный подход: Claude + YandexGPT + GigaChat + open-source

## 1. Анализ рынка

### Мировой рынок AI-разработки
- Глобальные расходы на AI: ~$490B к концу 2026
- Рынок agentic AI: $7.8B → $52B к 2030
- 72% организаций используют AI (McKinsey 2025)
- Средний проект: $120,594 / 10 месяцев (Clutch)

### Российский рынок
- Рынок AI в России: ~$2.1 млрд, рост 45%/год
- Генеративный AI: 58 млрд руб (x4.5 за год)
- 71% компаний уже применяют GenAI
- К 2033: до $40.67 млрд
- Федеральный проект «Искусственный интеллект»: 15.7 млрд руб на 2024-2026

### Спрос (по убыванию)
1. Автоматизация коммуникаций (чат-боты) — рынок 8.7 млрд руб
2. AI-агенты для бизнес-процессов — самый быстрорастущий сегмент
3. Генерация контента и маркетинг
4. Аналитика и прогнозирование
5. Computer Vision
6. RAG и корпоративные базы знаний

### Тренды 2025-2026
**Работает:**
- Multi-agent systems (специализированные агенты)
- MCP + A2A протоколы для интеграции агентов
- Domain-specific модели > универсальные LLM
- Value-based pricing (оплата по результату)

**Не работает:**
- Один LLM на все задачи
- AI без редизайна процессов (ROI ~7%)
- 40%+ agentic AI проектов будут отменены к 2027

## 4. Техническая архитектура

### Стек

| Слой | Технология | Почему |
|------|-----------|--------|
| Фронтенд | Next.js 15 + Tailwind + shadcn/ui | Зрелая AI-экосистема, SSG+SSR |
| AI Chat | Vercel AI SDK 6 + @ai-sdk/anthropic | Streaming, tool calling, structured output |
| Backend | Next.js Route Handlers | Fullstack, без отдельного сервера |
| БД | Supabase (PostgreSQL + Auth + Storage) | Бесплатный tier, realtime, RLS |
| PDF | @react-pdf/renderer | Генерация ТЗ и КП в PDF |
| CRM | Bitrix24 API (РФ) / HubSpot (бесплатный) | Интеграция с рос. рынком |
| Автоматизация | n8n (self-hosted) | Open-source, визуальные workflow |
| Email | Resend | Простой API, транзакционные письма |
| Деплой | Vercel (фронт) + VDS 185.209.30.232 (бэкенд) | Скорость + контроль |
| Мониторинг | Sentry + Vercel Analytics | Ошибки + метрики |

### Pipeline обработки заявки

```
Клиент заходит на сайт
  ↓
AI-чат виджет (Claude Sonnet через Vercel AI SDK)
  ↓
Tool calling: сбор требований по полям
  [тип проекта, фичи, дизайн, бюджет, сроки, контакты]
  ↓
Structured output → JSON (Zod-валидация)
  ↓
Генерация ТЗ (markdown → PDF)
  ↓
Оценка стоимости (Claude + прайс-лист в контексте)
  ↓
Генерация КП (PDF + веб-страница)
  ↓
CRM: создание сделки (Bitrix24/HubSpot)
  ↓
Уведомления: Telegram менеджеру + email клиенту с КП
  ↓
Календарь: предложение слота для созвона (Cal.com)
```

### AI-модели

| Задача | Основная модель | Fallback |
|--------|----------------|----------|
| Диалог с клиентом | Claude Sonnet 4.5 | YandexGPT 5 Pro |
| Генерация ТЗ/КП | Claude Opus 4.6 | Claude Sonnet 4.5 |
| Оценка стоимости | Claude Sonnet 4.5 | - |
| SEO-контент | YandexGPT 5 Pro | GigaChat 2.0 |

**Стратегия доступа к моделям:**
- Claude API: через зарубежную карту (Казахстан/Турция)
- YandexGPT: рублёвая оплата через Яндекс.Облако
- GigaChat: рублёвая оплата через Сбер
- Fallback: DeepSeek/Mistral на собственном сервере

## 5. Структура сайта

### Страницы

```
/ — Главная (hero + услуги + кейсы + AI-чат)
/services — Каталог услуг
/services/chatbot — AI чат-боты
/services/assistant — AI-ассистенты
/services/automation — Автоматизация
/services/website — Разработка сайтов
/services/consulting — AI-консалтинг
/portfolio — Портфолио / Кейсы
/portfolio/[slug] — Кейс
/about — О студии
/blog — Блог (SEO)
/blog/[slug] — Статья
/contact — Контакты
/brief — AI-сбор ТЗ (полноэкранный чат)
/estimate — Калькулятор стоимости
/client — Личный кабинет клиента
/client/projects — Проекты
/client/documents — ТЗ, КП, договоры
```

### Ключевые элементы главной

1. **Hero-секция:** заголовок + AI-чат виджет (CTA: «Опишите проект — получите оценку за 5 минут»)
2. **Услуги:** 6 карточек с иконками
3. **Как мы работаем:** 4 шага (заявка → ТЗ → разработка → запуск)
4. **Кейсы:** 3-4 карточки с цифрами
5. **Калькулятор:** интерактивный расчёт стоимости
6. **Блог:** последние 3 статьи
7. **CTA:** форма + AI-чат
